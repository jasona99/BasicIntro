Several steps must be followed to connect to the on-Grounds printers for Windows computers.
  • Press the Windows key and R to launch Run
  • Type ""\\pcs-vas.eservices.virginia.edu" without quotes into the box and click OK
  • Type "ESERVICES\mst3k" without quotes into the username in the security box that appears, where mst3k is your computing ID
  • Type your password for Eservices into the password box
  • This should be your NetBadge password
  • Select “Remember my credentials” and click OK
  • Find "PCS_Public_BW_Printers" and "PCS_Public_Color_Printers" in the window that opens, right click each of these, and choose connect
The public printers around Grounds will now be accessible when printing a document
