Several steps must be followed to connect to the on-Grounds printers for macOS computers.
  • Launch System Preferences, choose Printers and Scanners, then click + to add a printer
  • Go to Advanced
  • Choose "Windows printer via spoolss" for Type
  • For color printers, use the URL "smb://pcs-cas.eservices.virginia.edu/pcs_public_color_printers"
  • For black and white printers, use the URL "smb://pcs-cas.eservices.virginia.edu/pcs_public_bw_printers"
  • For Use, select "Generic PostScript Printer"
  • Click Add
  • Check the box next to Duplex Printing Unit
  • Repeat to add the unadded printer.
The public printers around Grounds will now be accessible when printing a document
