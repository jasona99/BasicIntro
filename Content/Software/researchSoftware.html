The major research software programs that Batten students use are:
  • Stata
    • Programmable statistics and data package
  • ArcGIS
    • Spatial analysis and planning tools
  • StatTransfer
    • Simplifies the transfer of data between different applications
  • SAS
    • Statistical analysis application
  • SPSS
    • Statistical Package for the Social Sciences
  • NVivo
    • Analysis of unstructured data
